Add pages

  • Use the add pages to add data to the database. When creating a new student you must add them to the class/cca/activity before being able to use the update/edit functions in the view and update pages
  • You would be unable to add records with the same name or in the case for subjects no same record with the same name and level.
  • You can change the data in the confirm page and the page will only add the data from the confirm page
  • You cannot add data containing "#", "&" and "="

View pages

  • The search bar searches through the list of records and formats the table to show only the records with text in the search input
  • Use the drop down box to change which field to search by
  • Click on the headers to sort the table by the field in ascending order
  • Hyperlinks in the students/ccas/activities/subjects headers will show another page with more detailed information on these fields
  • Use the edit button to edit the records, the edit functions the same as the update page
  • Use the delete button to remove records from the database
  • You cannot edit data containing "#", "&" and "="

Update pages

  • Records to be updated must exist
  • You cannot update records to have the same name as other records.
  • Only the data submitted on the confirm page is used.
  • You cannot edit data containing "#", "&" and "="
  • Updating student CCA membership lets you update the role of the student in the specified CCA he is in.
  • Updating student activity membership lets you update the role, category, award and hours of the student in the specified activity he is in.
  • Use add pages to add students to CCAs and activities.
